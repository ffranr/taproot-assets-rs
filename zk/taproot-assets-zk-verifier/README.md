# taproot-assets-zk-verifier

Zero-knowledge proof verifier for Taproot Assets.

This crate provides verification logic for ZK proofs generated by Taproot Assets
guest circuits. It is designed for use with proof systems like
[RISC Zero](https://www.risczero.com/) and integrates with the shared interfaces
in `taproot-assets-zk-core`.

## Usage

Add to your `Cargo.toml`:

```toml
taproot-assets-zk-verifier = { version = "0.0.1" }
```
